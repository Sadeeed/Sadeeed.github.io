CLI Tools

Homebrew

Having a package manager is a must for me, downloading apps from the internet is a pain and I prefer to do it from the terminal when possible. Homebrew is a package manager for macOS. It makes installing and updating packages a breeze. It also has a cask feature which allows you to install GUI applications.

Installing homebrew is as easy as running the following command in your terminal:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Note: I’ll be using homebrew to install most of the tools in this list.

Pyenv

Coming from Manjaro I am used to having multiple versions of python installed on my machine. But its not that easy to install and manage multiple Python installations on MacOS and many other linux distributions for that matter. That is where pyenv comes in. It allows you to install and manage multiple versions of python on your machine. It also allows you to set a global python version or set a python version for a specific directory. This is very useful when working on multiple projects that use different versions of python.

Install pyenv using homebrew:

brew install pyenv

We need to do a few more things to get pyenv working. (This is only for zsh, if you are using bash follow the instructions here

echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.zshrc
echo 'command -v pyenv >/dev/null || export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.zshrc
echo 'eval "$(pyenv init -)"' >> ~/.zshrc
brew install openssl readline xz tcl-tk

Nvm

For the same reason as pyenv I also use nvm to manage multiple versions of node.

nvm can be installed using their install script:

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash

Powerlevel10k

Manjaro comes with powerlevel10k pre-installed and I have been using it for a while now. I really like the way it looks and the features it provides. So I decided to install it on my mac as well.

Installation is pretty simple but we have to add it the our .zshrc file so it can load when we open a new terminal.

brew install romkatv/powerlevel10k/powerlevel10k
echo "source $(brew --prefix)/opt/powerlevel10k/powerlevel10k.zsh-theme" >>~/.zshrc

Before we proceed i would recommend installing a nerd font otherwise you will see weird symbols in your terminal. Thats it! Now you can run p10k configure to configure powerlevel10k or you can restart your terminal.

GUI Tools

Alacritty

I dont really like the default terminal that comes with macOS so I decided to install Alacritty. It is a terminal emulator with a lot of features and it is very fast. It is also very customizable and you can change the font, colors, etc. I use it with powerlevel10k and it looks great.

It is one of those few things that I’d recommend installing from the website not because the homebrew version is bad but because the website has a link to the short and concise docs which are very helpful. Anyway, if you prefer you can install it using homebrew:

brew install alacritty

The default icon did not fit so I replaced it with this one:

You can either download the icon from here or you can run the following command to download and set the icon:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/hmarr/dotfiles/main/bin/update-alacritty-icon.sh)"

Termius

Termius is the only SSH client that I have used and I really like it. It has a lot of great features and it is very easy to use. It also has a mobile app which is very useful when I am on the go. I use the free version which has all the features except sync between devices although I have found a way to sync my settings between devices which I will talk about in a later post. It also has FTP / SFTP support which makes it a great all in one tool.

Maccy

I had a clipboard manager extension installed on gnome and I really missed it on macOS. I tried a few different ones but I ended up using Maccy.

Download the latest version from github releases page, or use Homebrew:

brew install maccy

I also like the fact that it has a built a database viewer which has been very helpful when working with databases. I use the professional version at work but the community version is also great.
